# ES6 Classes

## Description
This project focuses on learning and practicing **ES6 classes** in JavaScript.  
It covers class definition, methods, static methods, inheritance, and an introduction to metaprogramming and symbols.

## Learning Objectives
By the end of this project, you should be able to:
- Define a class in ES6
- Add methods to a class
- Understand and use static methods
- Extend a class from another class
- Understand the basics of metaprogramming and symbols

## Requirements
- Ubuntu 20.04 LTS
- Node.js 20.x.x
- npm 9.x.x or higher
- Allowed editors: vi, vim, emacs, Visual Studio Code
- All files must end with a new line
- All JavaScript files must use the `.js` extension
