# Node JS Basics

## Description

This project introduces the fundamentals of Node.js and Express.  
By the end of this project, you will be able to run JavaScript with Node.js, use modules, handle files, work with HTTP servers, and use modern JavaScript (ES6) with Babel.

---

## Learning Objectives

At the end of this project, you are expected to be able to explain to anyone, without the help of Google:

- Run JavaScript using Node.js
- Use Node.js modules
- Use specific Node.js modules to read files
- Use `process` to access command line arguments and environment variables
- Create a small HTTP server using Node.js
- Create a small HTTP server using Express.js
- Create advanced routes with Express.js
- Use ES6 with Node.js using Babel-node
- Use Nodemon to develop faster
